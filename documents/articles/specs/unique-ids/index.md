# entrypoint: unique-ids

一意なIDを生成するためのユーティリティ関数を提供します。

## exported members

- [`getUUIDv7`](./getUUIDv7.md): UUIDv7を生成する関数

## Usage

- Web Browsers
    ```js
    // Adhoc import
    import { getUUIDv7 } from "https://esm.sh/jsr/@ayaexptech/arsenal/unique-ids";
    ```
- Deno Adhoc Import
    ```ts
    import { getUUIDv7 } from "jsr:@ayaexptech/arsenal/unique-ids";
    ```
