# function: CCW

2次元空間における3点`p0`, `p1`, `p2`の位置関係を判定します。
3点が反時計回りに配置されている場合は`1`、時計回りに配置されている場合は`-1`、一直線上にある場合は`0`を返します。

### `CCW(p0: Vector2D, p1: Vector2D, p2: Vector2D): 0 | 1 | -1`

#### 引数

- `p0: Vector2D`
    - 1つ目の点を表す2次元ベクトルです。
- `p1: Vector2D`
    - 2つ目の点を表す2次元ベクトルです。
- `p2: Vector2D`
    - 3つ目の点を表す2次元ベクトルです。

#### 戻り値

- `( -1 | 0 | 1 )`
    - 3点が反時計回りに配置されている場合は`1`を返します。
    - 3点が時計回りに配置されている場合は`-1`を返します。
    - 3点が一直線上にある場合は`0`を返します。

#### 例

```js
import { Vector2D, CCW } from 'path/to/geometry'
const p0 = new Vector2D(0, 0);
const p1 = new Vector2D(1, 1);
const p2 = new Vector2D(2, 0);
console.log(CCW(p0, p1, p2)); // -1 (時計回り)
```

#### 計算量

- 時間計算量
    - 最悪: $O(1)$
