# constructor: new Treap

`Treap`クラスのコンストラクタです。新しい`Treap`インスタンスを作成します。

## 構文

```js
new Treap(keyCompareFn)
```

### 引数

- `keyCompareFn: (a: K, b: K) => number`
    - キーを比較するための関数です。
    - 引数として、2つのキー`a`と`b`が渡され、`a`と`b`の順序関係を示す数値を返すことが要求されます。
        - `a`が`b`より先(数値昇順なら"未満")の場合は負の数を返します。
        - `a`が`b`より後(数値昇順なら"大きい")の場合は正の数を返します。
        - `a`と`b`(の順序関係)が等しい場合は0を返します。
    - 例: 数値の昇順で比較する場合、`(a, b) => a - b`のように定義します。

### 戻り値

- `Treap<K, V>`
    - 新しく作成された`Treap`インスタンスを返します。

## 例

```js
import { Treap } from 'path/to/treap';
// 数値の昇順でキーを比較する関数を渡してTreapインスタンスを作成
const treap = new Treap((a, b) => a - b);
```

## 計算量

- 時間計算量
    - 最悪: $O(1)$
