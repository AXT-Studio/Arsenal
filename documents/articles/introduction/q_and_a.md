# Q&A

## 使い方

:::details AyaExpTech Arsenalはどうやって使いますか？
[Installation](./installation.md)をご覧ください。
:::

:::details (任意の実行環境)で使えますか？
ECMAScript Moduleをサポートしていて、かつ外部モジュールをインポートできる環境であれば基本的に動作します。
そうでなくとも、公開されているコードを直接ダウンロード、もしくはコピペして利用することも可能です。
:::

## その他

:::details "AyaExpTech"って何？
AyaExpTechは、本ライブラリの作者（綾坂こと）の個人制作ブランドの名前です。
https://axtech.dev/
:::

:::details "AyaExpTech Arsenal"の読みは？
少なくとも"Arsenal"の部分は「アーセナル」と読みます。

AyaExpTechの読みは正式には定めていません。
私はもともと「アヤエクスプテック」と読んでいましたが、発音が面倒なので最近は「エーエックステック」と読むことが多いです。
:::
